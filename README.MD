# Compile & run
(Linux)
Run on terminal (with Dbclean)
``` 
> mvn clean dbclean:dbclean package exec:java
``` 
Run on terminal(without Dbclean)
``` 
> mvn clean package exec:java -Dexec.args="drive.xml"
```
